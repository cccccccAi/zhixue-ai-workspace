## 技能详情

| 技能               | 说明                                                   | 产出             |
| ------------------ | ------------------------------------------------------ | ---------------- |
| `/write-prd`       | 内置实体定义和指标字典，引导真实数据填充，20项交付质检 | 可提交评审的 PRD |
| `/data-analysis`   | 趋势/对比/归因/分布分析，内置教育数据验证清单          | 数据分析报告     |
| `/weekly-report`   | 输入本周进展，按读者（领导/研发/学校）生成不同口径周报 | 周报文档         |
| `/metrics-review`  | 输入指标数据，自动标注异常（🟢🟡🔴），生成复盘报告     | 复盘报告         |
| `/build-dashboard` | 输入数据或选预设，生成交互式 HTML 看板（Chart.js）     | HTML 看板        |

## Things That Will Bite You

1. **跳过数据填充** — /write-prd 的核心价值是引导你填入真实数据，跳过等于只拿到模板
2. **忽视产品背景** — 智学网是教育科技平台（考试系统+批阅机+个性化学习），技能已内置上下文
3. **忽略季节性** — 教育数据受学期节奏影响大（考试月、寒暑假），分析时优先用同比
4. **模拟数据当真** — 模拟数据仅演示方法论，不能作为决策依据

## 压缩规则

When compacting, always preserve:

- 用户提供的真实数据和场景
- learning/outputs/ 中已生成的文档列表
- 当前分析的主题和进度
