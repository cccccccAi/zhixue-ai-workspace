# 智学网指标层级与复盘框架

> 本文件定义智学网产品的指标层级和复盘分析框架。
> 实体和指标精确定义参考 domain/zhixue-domain.md。

## 指标层级

### North Star（北极星指标）

**周活跃考试学生数** = 本周至少完成 1 次考试的学生数

选择依据：

- 学生完成考试 = 产品核心价值交付（考试→批阅→学情报告）
- 与续费正相关：学生活跃 → 教师感知价值 → 学校续费
- 可拆解：受学校数、教师活跃度、考试频次共同驱动

### L1 指标（健康指标）

| 维度   | 指标           | 口径                                             | 健康基准     |
| ------ | -------------- | ------------------------------------------------ | ------------ |
| 获客   | 新签约学校数   | 本周/月新签合同的学校                            | [待填: 基准] |
| 激活   | 教师首次使用率 | 注册后 7 天内至少使用 1 次 / 新注册教师          | > 60%        |
| 活跃   | 教师周活跃率   | 本周至少操作 1 次的教师 / 注册教师（排除已离职） | > 40%        |
| 留存   | 学校月留存率   | 本月有活跃教师的学校 / 上月有活跃教师的学校      | > 90%        |
| 营收   | 学校续费率     | 续费学校 / 到期学校                              | > 75%        |
| 满意度 | 教师 NPS       | 净推荐值                                         | > 20         |

### L2 指标（诊断指标）

当 L1 异常时，按以下方向下钻：

| L1 异常          | 下钻方向（L2）                                                                     |
| ---------------- | ---------------------------------------------------------------------------------- |
| 教师活跃率下降   | 按角色拆分（任课/班主任/教研组长）、按学科拆分、按地区拆分、报表使用率、考试创建率 |
| 学校留存率下降   | 按学校规模拆分、按地区拆分、按功能使用深度拆分、续费前 3 个月活跃趋势              |
| 教师首次使用率低 | 注册渠道分析、引导流程完成率、首次使用功能分布                                     |
| 续费率下降       | 按合同金额拆分、续费前活跃度、竞品流失分析                                         |
| NPS 下降         | 按角色拆分、按功能满意度拆分、负面反馈关键词                                       |

## 异常判定规则

### 环比变化阈值

| 变化幅度   | 状态        | 含义                     |
| ---------- | ----------- | ------------------------ |
| ≥ +10%     | ⬆️ 显著提升 | 好消息，但确认是否可持续 |
| +5% ~ +10% | 🟢 正常上升 | 健康趋势                 |
| -5% ~ +5%  | 🟢 稳定     | 正常波动                 |
| -5% ~ -10% | 🟡 关注     | 连续 2 周则升级为 🔴     |
| ≤ -10%     | 🔴 异常     | 必须归因并制定行动       |

### 季节性修正

| 时期                | 影响                           | 判定调整                 |
| ------------------- | ------------------------------ | ------------------------ |
| 考试月（期中/期末） | 学生活跃、教师使用自然上升     | 用同比而非环比判断       |
| 寒暑假              | 所有指标自然下降 30-50%        | 不标为异常，与去年同期比 |
| 开学季（3月/9月）   | 新注册激增，首次使用率可能偏低 | 关注绝对值而非比率       |
| 续费季（Q3-Q4）     | 续费率是焦点指标               | 提高续费相关指标的关注度 |

## 复盘四步法

### 1. 数据（What）

- 本期各指标的实际值
- 与上期/目标/基准的对比
- 趋势方向（上升/下降/稳定）

### 2. 洞察（So What）

- 哪些指标异常？严重程度？
- 是偶发还是持续趋势？
- 与其他指标是否联动？（如教师活跃↓ + 续费率↓ = 关联）

### 3. 归因（Why）

- 产品变更：最近上线了什么？下线了什么？
- 外部因素：考试季？政策变化？竞品动作？
- 数据质量：是真的变化还是统计口径变了？

### 4. 行动（Now What）

- 每个异常指标对应 1-2 个具体行动
- 明确负责人和截止时间
- 区分"立即做"和"下个迭代做"

## OKR 关联

### 指标→OKR 映射

复盘时自动检查指标变化对 OKR 的影响：

```
如果 North Star 下降 → 检查 O1（产品价值）进度
如果教师活跃率下降 → 检查 KR（教师活跃相关）进度
如果续费率下降 → 检查 O2（商业增长）进度
```

### 季度 OKR 打分标准

| 完成度    | 评分   | 含义                            |
| --------- | ------ | ------------------------------- |
| 0.0 - 0.3 | 未达成 | 需要复盘为什么，调整策略        |
| 0.4 - 0.6 | 有进展 | 方向对但力度不够                |
| 0.7 - 1.0 | 达成   | 70% 是 stretch OKR 的理想完成度 |
| > 1.0     | 超额   | 目标可能不够有挑战性            |
