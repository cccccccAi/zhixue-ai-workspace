# 智学网指标复盘

## 执行协议

- 产品同行语气，不教基础
- **全部中文输出**
- 每次输出 ≤ 15 行（不含代码块和表格），超过时用 AskUserQuestion 过渡
- 产出文件命名：`metrics-review-{类型}-{日期}.md`，存入 `./learning/outputs/`
- md 文件生成后用 `open -a "MarkText"` 自动打开
- 不确定的数据用 `[待填: 描述]` 占位，严禁编造指标数值

---

## 启动流程

### 1. 解析输入

解析 `$ARGUMENTS`：

- **含"周"** → 直接进入周复盘流程
- **含"月"** → 直接进入月复盘流程
- **含"季"** → 直接进入季度复盘流程
- **空或其他** → 用 AskUserQuestion 选择复盘类型

### 2. 检测历史复盘

自动检测 `./learning/outputs/metrics-review-*.md`：

- **存在** → 读取最近一份，用于环比对比
- **不存在** → 首次复盘，标注为基线

### 3. 加载领域知识

读取 [zhixue-domain.md](../../domain/zhixue-domain.md) 获取指标定义和口径。

## 四步工作流

### Step 1：选择复盘类型

用 AskUserQuestion 问：

```
选择复盘类型：
- 周复盘（5 分钟，聚焦异常和行动）
- 月复盘（15 分钟，含趋势分析和深度归因）
- 季度复盘（20 分钟，含 OKR 进度和战略建议）
```

### Step 2：指标数据输入

用 AskUserQuestion 问：

```
指标数据怎么来？
- 我粘贴数据（表格/文字/数字都行）
- 先用框架，数据后面填（生成带占位符的报告）
- 用上次复盘做对比基线
```

**如果用户粘贴数据**：

- 接收任意格式（表格、文字、逗号分隔、截图描述）
- AI 自动整理成 North Star → L1 → L2 结构
- 无法归类的指标放在"其他指标"区

**如果用户选择"先用框架"**：

- 按 [metrics-framework.md](references/metrics-framework.md) 生成完整框架
- 所有数值用 `[待填: 指标名当前值]` 占位

### Step 3：自动分析

按 [metrics-framework.md](references/metrics-framework.md) 中的规则执行：

**异常检测**：

- 环比下降 ≥ 10% → 🔴 异常
- 环比下降 5-10% → 🟡 关注
- 环比变化 < 5% → 🟢 正常
- 环比上升 ≥ 10% → ⬆️ 显著提升（也需关注，确认是否可持续）

**归因建议**：

- 对每个 🔴/🟡 指标，自动生成 2-3 个可能原因
- 参考智学网季节性规律（考试月、寒暑假、续费季）
- 如果有历史复盘数据，对比趋势判断是偶发还是持续

**行动建议**：

- 对每个异常指标生成"建议动作 + 负责人 + 优先级"
- 行动要具体可执行，不要写"继续关注"

### Step 4：输出复盘报告

根据复盘类型选择模板，参考 [review-templates.md](references/review-templates.md)。

**周复盘** — 快速扫描：

```
指标速览表（North Star + L1）
异常标注（🔴/🟡 + 可能原因）
本周行动清单
下周关注点
```

**月复盘** — 深度分析：

```
指标全景（North Star + L1 + L2 完整）
趋势分析（近 4 周走势 + ASCII 趋势图）
深度归因（异常指标逐个分析）
行动计划（负责人 + 截止时间）
OKR 进度映射
```

**季度复盘** — 战略视角：

```
OKR 打分（0-1.0）
指标健康度全景（全部 L1 + 关键 L2）
同比/环比对比
成功经验 + 失败教训
下季度战略建议
```

**生成后**：

1. 用 Write 写入 `./learning/outputs/metrics-review-{类型}-{日期}.md`
2. 用 `open -a "MarkText"` 打开
3. 输出简要确认：异常指标数量、关键发现

## Anti-Patterns

- **不要编造指标数值**：用户没给的数据用占位符
- **不要只列数据不给洞察**：每个异常指标必须有归因和建议
- **不要忽略季节性**：考试月指标波动是正常的，要标注
- **不要给模糊建议**："继续关注"不是行动，要具体到"谁做什么"
- **不要一次展示所有 L2 指标**：周复盘只看 L1，L2 按需展开
