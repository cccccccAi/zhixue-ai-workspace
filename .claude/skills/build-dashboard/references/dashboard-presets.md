# 预设看板配置

> 按受众预设的看板配置。AI 根据用户选择的受众加载对应预设。

## 教师看板

**适用角色**：任课教师、班主任
**核心问题**：这次考试我班上情况怎么样？哪里薄弱？谁需要关注？

### KPI 卡片（4 个）

| 指标         | 数据源                    | 变化对比       |
| ------------ | ------------------------- | -------------- |
| 全班均分     | assessment 表             | 与上次考试对比 |
| 年级排名     | assessment 表（跨班计算） | 与上次对比     |
| 及格率       | 及格人数 / 总人数         | 与上次对比     |
| 最大进步学生 | 前后两次分差最大的学生    | —              |

### 图表（3 个）

1. **分数段分布**（柱状图）：0-59 / 60-69 / 70-79 / 80-89 / 90-100
2. **薄弱知识点 Top 5**（横向柱状图）：知识点名称 + 全班正确率
3. **近 4 次考试趋势**（折线图）：均分变化 + 年级均值参考线

### 筛选器

- 考试选择（下拉：最近 5 次考试）
- 学科选择（如果班主任视图）

### 数据表格

| 姓名 | 分数 | 年级排名 | 与上次分差 | 薄弱知识点 |
| ---- | ---- | -------- | ---------- | ---------- |

## 学校管理看板

**适用角色**：校长、教务主任
**核心问题**：全校教学质量怎么样？哪个年级/学科需要关注？教师们在用智学网吗？

### KPI 卡片（5 个）

| 指标         | 数据源                          | 变化对比       |
| ------------ | ------------------------------- | -------------- |
| 全校均分     | 全校学生成绩                    | 与上次考试对比 |
| 考试总场次   | 本月考试数量                    | 环比上月       |
| 教师使用率   | 活跃教师 / 注册教师             | 环比上月       |
| 报表查看率   | 查看报表的教师 / 发起考试的教师 | 环比           |
| 使用最少年级 | 活跃度最低的年级                | —              |

### 图表（4 个）

1. **各年级均分对比**（柱状图）：年级名称 + 均分 + 标注最高/最低
2. **各学科均分对比**（柱状图）：学科 + 均分
3. **教师使用趋势**（折线图）：近 6 个月教师活跃率
4. **考试频次趋势**（折线图）：近 6 个月月考试场次

### 筛选器

- 年级选择
- 时间范围（本学期 / 近 3 个月 / 近 6 个月）

### 数据表格

| 年级 | 班级 | 均分 | 最高分 | 及格率 | 任课教师 | 教师使用状态 |
| ---- | ---- | ---- | ------ | ------ | -------- | ------------ |

---

## 区域教育局看板

**适用角色**：区/市教育局管理者
**核心问题**：辖区内各校教学水平？智学网渗透率？

### KPI 卡片（4 个）

| 指标       | 数据源                         | 变化对比     |
| ---------- | ------------------------------ | ------------ |
| 辖区学校数 | 签约学校                       | 较上月       |
| 使用渗透率 | 活跃学校 / 签约学校            | 环比         |
| 区域均分   | 全区学生均分                   | 与上学期同期 |
| 考试覆盖率 | 至少考过 1 次的班级 / 全部班级 | 环比         |

### 图表（3 个）

1. **各校均分排名**（横向柱状图）：学校名 + 均分 + 区域均值参考线
2. **使用渗透趋势**（折线图）：近 6 个月
3. **各校教师活跃率**（柱状图）：学校 + 教师活跃率

### 筛选器

- 学段（小学 / 初中 / 高中）
- 时间范围

---

## 产品团队看板

**适用角色**：产品经理、运营
**核心问题**：产品健康度？哪个功能需要优化？用户在增长吗？

### KPI 卡片（5 个）

| 指标              | 口径                  | 变化对比 |
| ----------------- | --------------------- | -------- |
| WAU（周活跃用户） | 本周有操作的教师+学生 | 环比上周 |
| DAU/MAU           | 日活/月活比值         | 环比     |
| 功能使用 Top1     | 使用次数最多的功能    | —        |
| 续费率            | 续费学校 / 到期学校   | 环比     |
| 页面加载 P95      | 95 分位加载时间       | 环比     |

### 图表（4 个）

1. **DAU 趋势**（折线图）：近 30 天 DAU + 7 天均线
2. **功能使用分布**（环形图）：各功能使用占比
3. **新注册趋势**（柱状图）：近 12 周新注册学校/教师
4. **续费率趋势**（折线图）：近 6 个月续费率

### 筛选器

- 用户类型（教师 / 学生 / 管理员）
- 时间范围

### 数据表格

| 功能 | 周使用次数 | 周活跃用户数 | 环比变化 | 使用率 |
| ---- | ---------- | ------------ | -------- | ------ |

## 模拟数据生成规则

当用户选择"用模拟数据"时，按以下规则生成：

- **分数**：均值 70-80，标准差 10-15，范围 30-100
- **人数**：一个班 40-50 人，一个年级 6-12 个班
- **使用率**：教师 30-60%，学生 50-80%
- **续费率**：70-85%
- **知识点正确率**：40-90%，薄弱知识点 < 60%
- **趋势**：加入合理波动（±5%），不要完美直线

模拟数据必须在看板顶部标注：

```html
<div class="mock-data-banner">⚠️ 当前展示为模拟数据，仅供演示看板效果</div>
```
