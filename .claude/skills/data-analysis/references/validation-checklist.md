# 数据验证清单

> 分析报告发出前的质量检查清单。AI 在生成分析报告时自动执行这些检查。
> 参考 Anthropic data-validation 框架，适配智学网教育场景。

## 一、数据质量检查

### 1.1 来源确认

- [ ] 数据来源是否明确（哪个系统、哪张表、谁提供的）
- [ ] 数据提取时间是否记录
- [ ] 是否为最新数据（排除缓存/快照延迟）

### 1.2 完整性

- [ ] 缺失值比例是否合理（> 5% 需要说明处理方式）
- [ ] 时间序列是否连续（有无缺失日期/周）
- [ ] 关键字段是否完整（学生姓名、班级、分数等核心字段不应缺失）

### 1.3 时效性

- [ ] 数据截止时间是否在报告标题中标注
- [ ] 是否存在"部分周期"问题（如月中分析月度数据）
- [ ] 对比时间范围是否对齐（本月30天 vs 上月28天）

### 1.4 去重

- [ ] 是否有重复记录（同一学生同一考试出现两条）
- [ ] Join 操作是否导致行数膨胀（多对多关联）

---

## 二、计算验证

### 2.1 聚合逻辑

- [ ] 求和/平均/计数的维度是否正确
- [ ] 百分比的分子分母是否定义清晰
- [ ] 加权平均是否使用了正确的权重（如按班级人数加权均分）

### 2.2 分母一致性

- [ ] 对比指标的分母口径是否一致
  - 例："活跃率"在不同报告中是否都用"注册教师数"做分母
- [ ] 分母是否排除了不应包含的群体（已离职教师、已毕业学生）

### 2.3 日期对齐

- [ ] 环比对比的时间跨度是否一致
- [ ] 同比对比是否考虑了学期日历差异（今年和去年的考试月可能不同）
- [ ] 是否考虑了工作日/休息日差异

### 2.4 率值计算

- [ ] 百分比是否在 0-100% 范围内（排除计算错误）
- [ ] 增长率的基数是否合理（基数太小时增长率会很夸张）
- [ ] 复合增长率 vs 简单增长率是否用对

---

## 三、合理性校验

### 3.1 量级检查

| 指标       | 合理范围 | 异常信号        |
| ---------- | -------- | --------------- |
| 班级人数   | 30-55 人 | < 10 或 > 70    |
| 考试分数   | 0-满分   | 负数或超出满分  |
| 教师活跃率 | 10-80%   | > 95% 或 < 3%   |
| 续费率     | 50-95%   | > 100% 或 < 20% |
| 学校均分   | 50-90    | < 30 或 > 95    |
| 年级班级数 | 4-20     | < 2 或 > 30     |

### 3.2 趋势连续性

- [ ] 趋势是否有不合理的跳变（上周100，本周10）
- [ ] 如果有跳变，是否有合理解释（数据源切换、口径调整）
- [ ] 预测值是否在历史波动范围内

### 3.3 交叉验证

- [ ] 各部分加总是否等于整体（各班人数之和 = 年级总人数）
- [ ] 相关指标是否方向一致（教师活跃率上升 + 考试频次下降 → 需解释）
- [ ] 与已知事实是否矛盾

---

## 四、呈现检查

### 4.1 数字格式

- [ ] 百分比统一使用 X.X% 格式（一位小数）
- [ ] 人数使用整数，不带小数
- [ ] 大数字使用千分位（如 1,234）或缩写（如 1.2万）
- [ ] 金额标注单位（元/万元）

### 4.2 结论检查

- [ ] 每个结论都有对应的数据支撑
- [ ] 相关性没有被表述为因果关系
- [ ] 模拟数据已在显著位置标注
- [ ] 局限性和假设已说明

### 4.3 行动建议检查

- [ ] 建议是否具体可执行（而非"继续关注"这类空话）
- [ ] 建议是否与分析发现直接相关
- [ ] 是否标注了优先级和负责方向

## 五、教育数据常见陷阱

### 5.1 存活者偏差

**问题**：只分析当前在校学生/在职教师，忽略已转学/离职的群体。

**影响**：续费率、留存率会被高估；满意度偏高（不满意的已经走了）。

**应对**：明确说明分析对象范围，标注是否包含流失群体。

### 5.2 学期边界效应

**问题**：跨学期对比时，不同学期的起止时间不同，考试安排不同。

**影响**：环比可能出现假异常（上学期有期末考，本学期初没有）。

**应对**：优先用同比（与去年同期比），环比时标注学期节点。

### 5.3 考试难度不一致

**问题**：不同考试的难度系数不同，直接比较均分没有意义。

**影响**：均分下降可能是试卷变难，而非学生退步。

**应对**：看排名变化、及格率变化，或使用标准分（Z分数）对比。

### 5.4 班额差异

**问题**：不同班级人数差异大，直接用均分排名不公平。

**影响**：小班的均分波动天然更大（样本小）。

**应对**：标注班级人数，对小样本（< 20人）的结论加注"样本量小，结论仅供参考"。

### 5.5 教师角色混淆

**问题**：同一教师可能同时是任课教师和班主任，在统计时被重复计数。

**影响**：教师数量被高估，活跃率可能被低估。

**应对**：按教师唯一 ID 去重，角色作为标签而非分组依据。

### 5.6 时区与日切

**问题**：不同系统的日切时间可能不同（0:00 vs 学校作息时间）。

**影响**：晚自习的数据可能被归到不同的日期。

**应对**：确认数据源的日切规则，统一后再分析。
