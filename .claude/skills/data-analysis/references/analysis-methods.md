# 数据分析方法论

> 本文件定义 4 种分析方法论，每种都包含通用框架 + 智学网教育场景适配。
> AI 根据用户问题自动选择合适的分析方法。

## 一、趋势分析

### 适用场景

- "最近几个月教师活跃率怎么样？"
- "这学期考试频次有变化吗？"
- "续费率是在上升还是下降？"

### 分析框架

**1. 确定时间粒度**

| 分析周期     | 推荐粒度  | 智学网场景             |
| ------------ | --------- | ---------------------- |
| 近 7 天      | 日        | 考试周期内每日使用情况 |
| 近 1-3 个月  | 周        | 教师活跃率周趋势       |
| 近 6-12 个月 | 月        | 续费率、注册量月趋势   |
| 跨年对比     | 学期/季度 | 同比分析               |

**2. 时间序列拆解**

```
观测值 = 趋势 + 周期性 + 异常波动

趋势：长期方向（上升/下降/持平）
周期性：学期节奏（考试月上升、假期下降）
异常波动：需要归因的偏离
```

**3. 环比与同比**

- **环比**：与上一个周期比（本月 vs 上月）。适合看短期变化
- **同比**：与去年同期比（本月 vs 去年同月）。消除季节性影响
- **教育场景优先用同比**：因为学期节奏固定，环比容易被假期干扰

**4. 拐点识别**

- 连续 2 周同方向变化 ≥ 5% → 趋势确认
- 单周变化 ≥ 10% → 异常事件，需要归因
- 结合产品发布日历和学校日历交叉验证

### 智学网季节性修正

| 时期                | 影响                       | 修正方法                   |
| ------------------- | -------------------------- | -------------------------- |
| 考试月（期中/期末） | 活跃度自然上升 20-40%      | 用同比，不要标为"大幅增长" |
| 寒暑假              | 所有指标下降 30-50%        | 不标为异常，与去年同期比   |
| 开学季（3月/9月）   | 新注册激增，活跃率分母变大 | 区分新老用户分析           |
| 续费季（Q3-Q4）     | 续费相关行为集中           | 提高续费指标关注权重       |

## 二、对比分析

### 适用场景

- "各年级的使用情况差异大吗？"
- "数学和英语的成绩哪个更好？"
- "A 校和 B 校的教师活跃率对比"

### 分析框架

**1. 选择对比维度**

智学网常用对比维度：

| 维度 | 典型对比             | 注意事项               |
| ---- | -------------------- | ---------------------- |
| 年级 | 高一 vs 高二 vs 高三 | 高三可能因高考特殊     |
| 学科 | 语数英 vs 理化生     | 考试频次不同影响数据量 |
| 班级 | 同年级不同班         | 班额差异影响率值       |
| 地区 | 不同区/市            | 经济水平、政策差异     |
| 时间 | 本学期 vs 上学期     | 确保时间跨度一致       |
| 学校 | 不同签约学校         | 学校规模差异大         |

**2. 设定基准线**

- **内部基准**：全校/全区均值、历史最佳值
- **外部基准**：行业均值（如有）、竞品数据（如有）
- **无基准时**：用中位数作为参考，避免被极端值拉偏

**3. 判断差异显著性**

| 差异幅度 | 判定     | 行动                     |
| -------- | -------- | ------------------------ |
| < 5%     | 正常波动 | 不需要特别关注           |
| 5-15%    | 值得关注 | 记录，持续观察           |
| 15-30%   | 显著差异 | 需要归因                 |
| > 30%    | 极端差异 | 优先调查，可能有数据问题 |

**4. 辛普森悖论提醒**

> 整体趋势可能与分组趋势相反。

示例：全校均分上升，但每个年级均分都在下降——原因是高分年级（高一）人数占比增加了。

**应对**：对比分析时，同时看整体和各子群，如果方向矛盾，要解释原因。

## 三、归因分析

### 适用场景

- "为什么上月教师活跃率下降了？"
- "续费率下降的原因是什么？"
- "为什么这次考试全校均分异常低？"

### 分析框架

**1. 构建假设树**

```
指标异常
├── 产品因素
│   ├── 功能变更（上线/下线/改动）
│   ├── 稳定性问题（报错、加载慢）
│   └── 用户体验退化
├── 外部因素
│   ├── 学校日历（考试周、放假）
│   ├── 政策变化（教育政策调整）
│   ├── 竞品动作（竞品促销、新功能）
│   └── 人员变动（教师离职、换校）
└── 数据因素
    ├── 统计口径变化
    ├── 数据采集异常
    └── 样本偏差
```

**2. 逐一验证**

对每个假设，找数据支撑或反驳：

| 假设         | 验证方法                         | 判定                |
| ------------ | -------------------------------- | ------------------- |
| 功能变更导致 | 检查变更时间线是否与指标拐点吻合 | 时间吻合 = 可能性高 |
| 假期影响     | 同比去年同期是否也有类似下降     | 同比一致 = 季节性   |
| 数据采集问题 | 检查原始数据量、埋点状态         | 数据缺失 = 数据因素 |

**3. 排序输出**

按可能性从高到低排列：

```
🔴 高可能性（有数据支撑）：[具体原因]
🟡 中可能性（部分证据）：[具体原因]
⚪ 低可能性（无直接证据）：[具体原因]
```

### 常见归因误区

| 误区           | 说明                                  | 正确做法                 |
| -------------- | ------------------------------------- | ------------------------ |
| 时间巧合当因果 | A 和 B 同时发生 ≠ A 导致 B            | 找多个证据交叉验证       |
| 只看单一原因   | 复杂问题通常有多个原因                | 评估各因素贡献权重       |
| 存活者偏差     | 只分析还在用的用户                    | 把流失用户纳入分析       |
| 平均数掩盖差异 | "均值正常"但两极分化                  | 看分布，不只看均值       |
| 忽略基数效应   | 从 2% 到 4% 是"增长 100%"但绝对值很小 | 同时报告绝对值和相对变化 |

## 四、分布分析

### 适用场景

- "这次考试成绩分布怎么样？"
- "教师使用频次是什么分布？"
- "学校规模分布是否均匀？"

### 分析框架

**1. 描述性统计**

必报指标：

| 统计量  | 用途                       | 智学网场景              |
| ------- | -------------------------- | ----------------------- |
| 均值    | 整体水平                   | 全班/全校均分           |
| 中位数  | 典型水平（不受极端值影响） | 教师典型使用频次        |
| 标准差  | 离散程度                   | 班级成绩分化程度        |
| P25/P75 | 中间 50% 的范围            | 成绩主体分布区间        |
| P5/P95  | 极端值边界                 | 特别优秀/特别薄弱的学生 |

**2. 分段分析**

智学网常用分段：

| 数据类型   | 分段方式                                                  |
| ---------- | --------------------------------------------------------- |
| 考试成绩   | 0-59 / 60-69 / 70-79 / 80-89 / 90-100                     |
| 教师活跃度 | 未使用 / 低频（月1-3次）/ 中频（周1-3次）/ 高频（日使用） |
| 学校规模   | 小型（<30班）/ 中型（30-60班）/ 大型（>60班）             |
| 使用时长   | <1分钟 / 1-5分钟 / 5-15分钟 / >15分钟                     |

**3. 分布形态判断**

| 形态         | 特征               | 常见场景             | 关注点               |
| ------------ | ------------------ | -------------------- | -------------------- |
| 正态分布     | 钟形，对称         | 大型考试成绩         | 关注标准差           |
| 右偏（长尾） | 多数低值，少量高值 | 教师使用频次         | 关注中位数和长尾用户 |
| 双峰         | 两个峰值           | 分层教学班级成绩     | 拆分为两组分析       |
| 均匀分布     | 各段差不多         | 少见，可能是数据问题 | 检查数据采集         |

**4. 异常值识别**

- **IQR 方法**：低于 P25 - 1.5×IQR 或高于 P75 + 1.5×IQR
- **业务阈值**：教育场景中，成绩 < 30 分或 > 100 分明显异常
- **处理方式**：标记但不自动剔除，让用户决定

## 五、教育场景分析专区

### 考试成绩分析

**标准分析维度**：

1. 整体概况：均分、中位数、及格率、优秀率
2. 分数段分布：各段人数和占比
3. 学科对比：各科均分横向对比
4. 班级对比：各班均分 + 与年级均值的差距
5. 进退步分析：与上次考试的分差排名
6. 知识点分析：各知识点正确率，识别薄弱环节

### 教师使用率分析

**标准分析维度**：

1. 活跃教师数 / 注册教师数 = 使用率
2. 按角色拆分：任课教师 vs 班主任 vs 教研组长
3. 按学科拆分：哪个学科教师使用最多/最少
4. 功能使用分布：组卷、阅卷、报表查看的占比
5. 使用深度：只登录 vs 实际使用核心功能

### 学校续费预测

**关键预测指标**：

| 指标           | 与续费的关系 | 预警阈值                 |
| -------------- | ------------ | ------------------------ |
| 教师月活跃率   | 正相关       | < 20% 高风险             |
| 考试频次       | 正相关       | 连续 2 月无考试 = 高风险 |
| 报表查看率     | 正相关       | < 10% 高风险             |
| 管理员登录频次 | 正相关       | 月 < 2 次 = 中风险       |

### 学情诊断

**分析步骤**：

1. 学生成绩趋势（近 3-5 次考试）
2. 各知识点掌握度（正确率矩阵）
3. 与班级/年级均值的偏差
4. 薄弱知识点聚类（找到共性问题）
5. 个性化提升建议（按学生分层）
