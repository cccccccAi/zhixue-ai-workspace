# 智学网 AI 工作台

智学网产品团队的 AI 技能工具箱。5 个即用技能，帮你用 AI 完成产品、数据和项目管理的核心工作。

## 这是什么

这不是一个需要学习的框架，也不是一套需要配置的工具链。它是一个打开就能用的 AI 工作台：

- **学习路径**：6 关闯关教学，从整理指标体系到创建团队专属 AI 技能
- **产品规划**：PRD 生成器，内置智学网的实体定义、指标字典和质检清单
- **项目管理**：干系人周报，按读者自动调整口径和详细程度
- **数据分析**：指标复盘 + 交互式数据看板，支持多种预设受众

基于 [Claude Code](https://docs.anthropic.com/en/docs/claude-code) 的 Skill 机制，所有知识和流程都编码在 markdown 文件中。

## 前提条件

1. 安装 Claude Code（[安装指南](https://docs.anthropic.com/en/docs/claude-code/getting-started)）
2. 确保有 Claude 账号（Max 订阅或 API Key）

## 快速开始

```bash
# 1. 克隆仓库
git clone https://github.com/cccccccAi/zhixue-ai-workspace.git

# 2. 进入目录
cd zhixue-ai-workspace

# 3. 启动 Claude Code
claude
```

启动后会自动显示可用技能列表。

## 技能介绍

### /learn-pm-data — AI 产品力训练营

6 关逐步解锁，每关约 15 分钟：

| 关卡  | 主题         | 产出                          |
| ----- | ------------ | ----------------------------- |
| 第1关 | 指标体系整理 | 指标框架文档 + 实战提示卡     |
| 第2关 | 用研综合实战 | 用研综合报告 + 实战提示卡     |
| 第3关 | 竞品分析     | 竞品分析简报 + 实战提示卡     |
| 第4关 | 数据叙事     | 数据叙事文档 + 实战提示卡     |
| 第5关 | 数据看板     | 交互式 Dashboard + 实战提示卡 |
| 第6关 | 创建AI技能   | 智学网定制数据分析技能        |

每关支持真实数据输入，也提供智学网模拟数据。随时可以中断，进度自动保存。

### /write-prd — PRD 生成器

5 步生成可交付评审的 PRD：

1. **需求锚定**：驱动力、时间约束、读者
2. **素材提取**：自动读取闯关训练产出
3. **骨架生成**：12 章节 PRD，真实数据用占位符标记
4. **数据填充**：引导逐项补充真实数据
5. **交付质检**：20 项检查，阻塞项必须修复

内置智学网领域知识：实体关系、数据表结构、指标精确定义、角色术语。

### /weekly-report — 干系人周报

输入本周做了什么，自动生成面向不同读者的周报：

| 读者版本 | 风格           | 重点内容                          |
| -------- | -------------- | --------------------------------- |
| 领导版   | 精炼、数据驱动 | 成果 → 指标变化 → 风险 → 下周重点 |
| 研发版   | 具体、可执行   | 需求变更 → 优先级 → 阻塞项        |
| 学校版   | 简洁、友好     | 新功能 → 操作指引 → 已知问题      |

### /metrics-review — 指标复盘

输入指标数据，自动按 North Star → L1 → L2 层级整理，标注异常状态：

- **周复盘**：快速扫描，聚焦异常指标和行动项
- **月复盘**：含趋势分析、深度归因、OKR 进度
- **季度复盘**：含同比/环比、OKR 打分、战略调整建议

### /build-dashboard — 数据看板

生成交互式 HTML 看板，单文件无需服务器，可直接发给学校或领导：

| 预设看板   | 核心指标                              |
| ---------- | ------------------------------------- |
| 教师看板   | 均分、排名、薄弱知识点、分数段分布    |
| 学校管理   | 各年级均分、教师使用率、考试频次      |
| 区域教育局 | 学校排名、区域均分、使用渗透率        |
| 产品团队   | DAU/MAU、功能使用率、续费率、加载性能 |

基于 Chart.js，支持筛选器、KPI 卡片、多种图表、响应式布局和打印。
